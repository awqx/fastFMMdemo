# fastFMMdemo: Example code to apply `fastFMM`

We provide this code to help readers reproduce figures and experiments in our manuscript.

## Introduction

The R Markdown files are separated based on the figure numbers in the manuscript and titled by number and a shorthand reference. 

- `data/`: Data, though some data files needed to run the code as-is may be missing. 
- `fun/`: Functions to support the R markdown files.
- `img/`: Contains image outputs with sub-folders named consistently with the R markdown files.
- `renv/`: Package management
